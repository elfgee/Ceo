# 용어집 (Glossary)

이 문서는 CEO 프로젝트에서 사용하는 주요 용어와 정의를 정리합니다. 모든 PRD 및 기획 문서 작성 시 이 용어집의 정의를 최우선으로 사용해야 합니다.

---

## 사용자 관련

### 회원
- **정의**: 중개사무소 대표
- **설명**: CEO 사이트를 사용하는 주체로, 중개사무소를 대표하여 매물 광고를 등록하고 관리하는 사용자
- **호칭 규칙**:
  - **메뉴명, 탭명 등 구조적 요소**: "회원" 사용
  - **설명 문구, 안내 메시지 등 사용자 대면 텍스트**: "중개사님" 사용 (더 친근하고 정중한 느낌)
- **참고**: "사용자" 대신 "회원" 또는 "중개사님" 용어 사용

---

## 매물 관련

### 매물
- **정의**: 중개사무소가 등록하는 부동산 아이템
- **설명**: 아파트, 원룸, 오피스텔, 빌라 등 회원이 광고 등록하는 부동산 정보
- **참고**: "아이템", "상품" 대신 "매물" 용어 사용

### 광고 등록
- **정의**: 매물을 CEO 사이트에 등록하는 행위
- **설명**: 회원이 매물 정보를 입력하여 시스템에 등록하는 과정
- **참고**: "등록", "업로드" 대신 "광고 등록" 용어 사용

### 검수
- **정의**: 등록한 광고가 허위가 아닌지 확인하는 프로세스
- **설명**: 직방의 직원 또는 시스템이 회원이 등록한 매물 광고의 진위 여부를 확인하는 과정
- **참고**: 검수는 광고 등록 후 진행되는 필수 프로세스입니다

### 검수 반려
- **정의**: 검수 과정에서 허위로 판단되어 반려된 상태
- **설명**: 검수 결과 허위 광고로 판단된 경우, 해당 광고가 "검수 반려" 상태로 지정됨
- **참고**: "반려", "거절" 대신 "검수 반려" 용어 사용
